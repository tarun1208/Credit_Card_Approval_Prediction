# 📌 Project Overview

Financial institutions receive thousands of credit card applications every day. Manually evaluating each application is time-consuming, expensive, and prone to human error.

This project automates the credit approval process using **Machine Learning**. By analyzing customer information such as income, education, employment, housing, and family status, the application predicts whether the applicant is likely to receive credit approval.

The project combines:

- Machine Learning
- Data Preprocessing
- Feature Engineering
- Flask Web Development
- Professional Frontend UI
- Model Deployment Ready Architecture
