About This Project

AI Based Credit Card Approval Prediction

A Machine Learning powered web application that predicts whether a customer is eligible for a credit card by analyzing demographic, financial and employment information. The project demonstrates a complete end-to-end Machine Learning pipeline from data preprocessing to deployment using Flask.

Project Objective

The objective of this project is to assist banks and financial institutions in making faster and more consistent credit approval decisions using Artificial Intelligence. Instead of manually evaluating every application, the trained Random Forest model predicts whether a customer is likely to receive approval based on historical credit records.

Business Value

  • ✔ Faster customer screening
  • ✔ Reduced manual effort
  • ✔ Consistent decision making
  • ✔ AI assisted credit approval
  • ✔ Reduced financial risk
  • ✔ Better customer experience

Machine Learning Pipeline

The project follows a complete end-to-end Machine Learning workflow, starting from raw data collection and ending with deployment as a Flask web application.

1. Data Collection

Imported the Credit Card Application Dataset and Credit History Dataset.

2. Data Cleaning

Handled missing values, removed invalid records, and treated outliers for reliable model training.

3. Feature Engineering

Created AGE, EMPLOYED_YEARS, INCOME_PER_PERSON and CHILD_RATIO features.

4. Encoding

Converted categorical features into numerical values using Label Encoding.

5. Feature Scaling

StandardScaler was applied to normalize numerical features before model training.

6. SMOTE

Balanced the highly imbalanced target classes using Synthetic Minority Oversampling Technique.

7. Model Training

Trained Logistic Regression, Decision Tree, Random Forest and Gradient Boosting models.

8. Hyperparameter Tuning

Optimized the Random Forest model using RandomizedSearchCV to improve performance.

9. Deployment

Saved the trained model and deployed it as an interactive Flask web application.

Workflow Summary

The complete pipeline begins with collecting raw customer application and credit history data, followed by preprocessing, feature engineering, categorical encoding, feature scaling, and handling class imbalance using SMOTE. Multiple machine learning algorithms were evaluated, with the Random Forest model selected as the final model after hyperparameter tuning. The trained model was then serialized using Joblib and integrated into a Flask web application to provide real-time credit card approval predictions through an intuitive user interface.

Why Random Forest?

Among all evaluated algorithms, the Random Forest classifier achieved the highest overall performance after hyperparameter tuning. It provided the best balance between classification accuracy, robustness, and generalization on unseen data. The optimized model reached an accuracy of 97.60% and an AUC score of 0.7265, outperforming Logistic Regression, Decision Tree, and Gradient Boosting. Therefore, the tuned Random Forest model was selected as the final model for deployment in the Flask application.
